# Sistema de Gestión de Empleados con EF Core, Búsqueda y Paginación
Nombre: Ian Moraga  
Carnet: C35281

Para usar la aplicacion, tienes las opciones de agregar un nuevo empleado, buscarlo, editarlo e incluo darlo de baja. 
La aplicación implementa paginación en la vista de empleados. Por defecto se muestran 5 registros por página, y el usuario 
puede avanzar o retroceder entre páginas. Se indica el total de registros y el número de páginas disponibles.

Ejemplo: https://localhost:7147/?busqueda=TI
